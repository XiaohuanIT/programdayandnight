#### 管道符号

- "｜" :将前一个命令执行结果传递给后面的命令

#### 重定向符号

- 输入重定向 "<"

````

read var < /path/to/a/file

echo $var 则显示/path/to/a/file文件的内容

````

- 输出重定向 ">(覆盖)",">>(追加)","2>(收集错误)"","&>（全部重定向到指定文件,不管错误与正确）"
